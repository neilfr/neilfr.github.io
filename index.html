Hearty Diet

Diet planning application for heart failure sufferers that need to control their potassium intake

Hearty Diet

Problem

Heart failure management medications can signifcantly affect potassium. While this requires meal planning in order to maintain appropriate levels of the nutrient, there are few easy-to-access tools to help.

Solution

An easy to use application that utilizes the Health Canada Nutrient database to select favourite foods, assemble them into meals and create daily meal plans that meet potassium nutrient consumption targets.

Technology

Health Canada Nutrient database, Javascript, MongoDB, Express, React, Node, bcrypt Authentication, MDB Bootstrap, axios, MVC design pattern

Site: Hearty Diet
Github: https://github.com/neilfr/project3
